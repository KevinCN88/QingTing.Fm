#QingTing.Fm是调用[蜻蜓FM](https://www.qingting.fm/)API 查询界面内容，进行在线播放。  
# <h3>运行环境</h3>

* Visual Studio 2019，dotNet Framework 4.6.1 SDK
* 支持Windows Win7、8、10  （如电脑没有m4a的解码器会播放不了）
